# channel_breakout_long

Single-leg breakout strategy with EMA trend gating and optional shock filtering.

Parameter keys:

- `trade_size`
- `length`
- `ema_period`
- `entry_buffer_ticks`
- `trade_direction`
- `use_shock_filter`
- `shock_atr_window`
- `shock_max_gap_atr`
- `shock_max_range_atr`
- `shock_max_close_change_atr`
